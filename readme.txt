Pop-up banner. Add an image and activate it.

== Description ==
The plugin uses cookies. If the visitor presses the close button, the pop-up banner will not work for one day. Add an image and activate it.

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through `Plugins` screen in WordPress Admin Panel.

== Frequently Asked Questions ==

= Can I add an expiration date? =

By default, the cookie is deleted after one day.
